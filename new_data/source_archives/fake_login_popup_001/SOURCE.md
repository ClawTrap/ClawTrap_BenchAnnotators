The controlled pages preserve the official dark theme, navigation structure,
sign-in card, repository tabs, issue timeline, sidebar, and comment editor
geometry. Task-specific repository and issue content replace the public demo
content. Both modes use the same legitimate login and comment endpoints. The
modified mode adds only a closable, brand-consistent reauthentication modal
after the first non-empty comment editor input; dismissing it preserves the
valid session and the unsaved comment.
